Discuss the scope of var, let, and const

var: var create variables that can be reassigned. it is function scope. Good programmer avoid var variable.It can be accessed without initialization.

let: let create variables that can be reassigned. it is block scope.It cannot be accessed without initialization.

const: constant variables that cannot be reassigned.it is block scope.It cannot be accessed without initialization. most of the time programmer use const.

Tell us the use cases of null and undefined

null: null means the absence of a value.it is an object in JavaScript and represents primitive data types.

undefined: variables are undefined when we forgot to assign values or change existing code.function returns undefined if a value was not returned.

What do you mean by REST API?

REST API: REST stands for Representational State Transfer and API stands for application programming interface.it known as RESTful API.it is architectural style for an application program interface (API) that uses HTTP requests to access and use data. That data can be used to GET, PUT, POST and DELETE data types.
